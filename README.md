# SuelCakes — Loja Online de Bolos e Doces

Este é um site moderno e responsivo para a **SuelCakes**, oferecendo informações sobre produtos, serviços e contato.  
O objetivo é apresentar o negócio online, permitir que clientes conheçam os produtos e entrem em contato para pedidos e encomendas.

---

##  Tecnologias Utilizadas
- **HTML5** — Estrutura do site  
- **CSS3** — Design, cores, layout responsivo  
- **JavaScript** — Funcionalidades interativas  
- **Git & GitHub** — Versionamento do projeto  
- **GitHub Pages** — Hospedagem gratuita do site  

---

## Como o Projeto Foi Criado

1. Criada a estrutura base em HTML:
   - Cabeçalho com menu de navegação
   - Secção Home com banner e destaque dos produtos
   - Secção Sobre a empresa
   - Secção Produtos/Serviços
   - Secção Contacto com formulário de contato
   - Rodapé com informações de contato e redes sociais

2. Estilização completa com CSS:
   - Layout responsivo para desktop e dispositivos móveis
   - Paleta de cores e fontes personalizadas
   - Efeitos visuais para botões e imagens

3. Funcionalidades JavaScript:
   - Validação básica do formulário de contato
   - Interações visuais (hover, scroll, etc.)

4. Testes locais realizados antes da hospedagem para garantir responsividade e funcionamento do formulário.

##  Testando Localmente

### ✔️ Método 1 — Abrir diretamente no navegador
- Abrir o arquivo `index.html` com dois cliques.

### ✔️ Método 2 — Usando VS Code + Live Server
1. Instalar extensão **Live Server**
2. Clicar com botão direito sobre `index.html`
3. Selecionar **Open with Live Server**

### ✔️ Método 3 — Android (Web Server for Android)
1. Colocar os arquivos na pasta `www`
2. Iniciar o servidor
3. Acessar pelo navegador do telemóvel

## 🌐 Hospedagem no GitHub Pages

### 1️⃣ Criar o repositório
- GitHub → **New Repository**
- Nome: `SuelCakes`
- Público → Create repository

### 2️⃣ Enviar os arquivos do site
- Add file → Upload files  
- Enviar todos os arquivos → Commit changes

### 3️⃣ Ativar o GitHub Pages
1. Ir em **Settings**  
2. Menu lateral → **Pages**  
3. Seção **Build and Deployment**:
   - Source → **Deploy from a branch**
   - Branch → **main**
   - Folder → **root**  
4. Salvar

### 4️⃣ Pegar o link gerado
O GitHub cria o link:

https://anatovele.github.io/SuelCakes/
